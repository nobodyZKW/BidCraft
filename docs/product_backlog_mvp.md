# BidCraft AI 产品代办表（MVP）

## 1. 使用说明
1. 本代办表仅覆盖 MVP 范围：`goods` + `public_tender`。
2. 所有条目都对齐当前仓库约束：模型输出必须通过 JSON Schema；存在高风险时禁止正式版导出。
3. 状态定义：`已完成`、`进行中`、`待开始`、`阻塞`。

## 4. 里程碑通过标准（建议）
1. 工程门禁通过：`pytest backend/tests -q`、`npm run build`、`npm run lint` 全通过。
2. 业务门禁通过：正式版阻断规则准确，one-shot 降级行为准确。
3. 演示门禁通过：完整 E2E 成功一次，高风险阻断一次，均有记录。
