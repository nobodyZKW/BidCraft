# BidCraft AI 风险追踪表（MVP）

## 1. 评估口径
1. 评分规则：`风险等级 = 概率 * 影响`，并结合业务敏感度人工校正。
2. 概率等级：`高(>60%)`、`中(30%~60%)`、`低(<30%)`。
3. 影响等级：`高`（影响核心交付/合规）、`中`（影响效率/质量）、`低`（可容忍）。
4. 状态：`开放`、`跟进中`、`已缓解`、`已关闭`。

## 3. 本周优先处理建议（按紧急度）
1. R-001 明文密钥。
2. R-002 前端 lint 自动化门禁。

## 4. 跟踪节奏建议
1. 每日站会：只看 `严重/高` 风险，更新时间与责任人。
2. 每周评审（周五）：更新概率、影响、关闭日期，清理过期风险。
3. 里程碑前 1 天：逐条确认 `R-001/R-002/R-013` 是否关闭，否则不建议对外演示。
